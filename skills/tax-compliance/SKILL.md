# 美股投资税务与合规手册

## 为什么存在

项目之前**完全没讨论过税务**——这是最惊险的短板。
美股红利、跨境结汇、遗产、CRS 信息交换每一项都可能直接吃掉 10-30% 收益。
本 skill 是查询手册 + 定期任务触发器。

## 核心事实（持有美股的中国居民）

### 1. 美国红利预扣税：30% → 10%（W-8BEN）

- **默认情况**：美国对非美居民持股红利按 **30%** 预扣
- **中美税收协定**：签 W-8BEN 后可降至 **10%**
- **适用到**：所有美股分红（持仓 ETF/个股）
- **节省金额估算**：假设年分红率 1.5%，组合 1000 万 → 年分红 15 万，税差 20% = **省 3 万/年**

### 2. 美国资本利得税：**非美居民免征** ✅

- **关键优势**：中国公民作为非美居民外国人（NRA），**卖美股的资本利得美国不征税**
- 前提：不是美国 real property（股票 ETF 不是）
- 持有期无所谓（短期/长期都免）
- **这是隐藏红利**：你赚 1000 万差价，美国一毛钱税不收

### 3. 中国境外投资所得：理论 20%，执法不严

- 《个人所得税法》：境外所得按"利息股息红利所得"或"财产转让所得"20% 征税
- 实际：个人小额投资执法稀疏，但**不是"合法避税"**
- 合规建议：超过一定规模（>千万）建议咨询税务师

### 4. Wash Sale Rule（亏损收割规则）

- 美国规定：卖出亏损股 → 30 天内买回**相同或实质相同**证券 → 亏损不能抵税
- 对我们影响：**几乎无影响**（不做 tax-loss harvesting，每月定投是"赚钱买入"）
- 警惕场景：如果未来想用亏损抵扣红利税，要避开 30 天窗口

### 5. 跨境结汇限额

- 中国个人每年结汇额度：**$50,000** / 人（2026 年规则，注意政策变化）
- 境外资金汇入：同样限额
- 港汇丰 One → 人民币：走"个人结汇"会占用额度
- 替代方案：
  - 保持美元不结汇（长期看美元是优质资产）
  - 小额多次
  - 家人额度分散（夫妻 2 人 = $100,000/年）

### 6. CRS 信息交换

- 中国 2017 加入 CRS，与 100+ 国家/地区交换金融账户信息
- 港汇丰 One 账户（香港）→ 信息会交换给中国税务
- **门槛**：个人账户余额 > HKD 100 万 = 高净值标签
- 合规核心：**如实申报境外所得**

### 7. 美国遗产税（关键长期风险）

- 非美居民持有美国资产 > **$60,000** 时身故 → 最高 **40%** 遗产税
- 当前你组合（市值视个人情况）**尚未触及**或**已触及**——按实际计算
- **若长期定投**（预估 N 年后达 ~$28 万）→ **严重触发**
- 对冲方案：
  - **ETF 注册地选择**：爱尔兰/卢森堡注册的 UCITS ETF（如 CSPX 替代 VOO）不适用美国遗产税
  - 用信托 / joint account
  - 持有 ADR 而非直接美股（部分 ADR 不算美国资产）
- **待办**：2030 年前研究 UCITS 替代方案

## 定期任务触发器

### 每年 3 月（W-8BEN 检查）

```
检查项：
1. <券商> 账户 "Tax Center" / "Account Settings"
2. W-8BEN 状态：Active？有效期到哪年？
3. 是否有待签署的续签通知？
4. 红利实际扣税比例（查去年 1099-DIV）：应是 10% 不是 30%
```

**W-8BEN 有效期：3 年**（交表日起算）
- 首次签署：YYYY-MM（<券商> 开户时）
- 到期日：**YYYY-MM**（首签 + 3 年）
- 每年 3 月 SOP 自动提醒：剩余 X 年到期

### 每次红利到账后

- 查看 <券商> "Confirmations"
- 确认扣税比例是 10% 不是 30%
- 如果是 30% → **W-8BEN 失效或未生效**，立即重新提交

### 每年 1 月（1099-DIV 汇总）

- <券商> 会发 1099-DIV 表（前一年分红汇总）
- 记录：
  - 总分红金额（$）
  - 已扣税金额（$）
  - 实际扣税率（应 ~10%）
- 写入 `investment/records/decision-journal.md` 年度税务复盘段

### 每次大额跨境结汇前

- 查当年已用额度（去国家外汇管理局 app 查"个人购汇结汇额度"）
- 如接近 $50,000，分批或推迟到次年
- 申报用途：建议"留学/境外投资"等正当理由

### 资产达 $60,000 时（遗产税警戒线）

**预估时点：达到时（按个人定投速度估算）**

到达时必须：
1. 研究 UCITS ETF 替代方案（如 CSPX 替代美国注册的大盘 ETF）
2. 考虑开联名账户或信托
3. 咨询跨境税务师

## 常见问题快速查

| 问题 | 答案 |
|---|---|
| 卖美股赚 50% 美国收税吗？ | **不收**（非美居民资本利得免税） |
| 卖美股赚 50% 中国收税吗？ | 理论 20%，执法宽松 |
| 红利被扣 30%！ | W-8BEN 未生效，去 <券商> 重签 |
| 能结汇多少？ | $50,000/人/年 |
| 美元转回人民币税？ | 结汇本身无税，但超额度违规 |
| 身故资产怎么办？ | **>$60k 有美国遗产税，用 UCITS ETF 规避** |
| 要不要报 CRS？ | 你不用主动报，银行自动报送 |
