# Growth Analysis

## Overview
Comprehensive growth analysis focusing on revenue, earnings, and margin trends.

## Analysis Process

### Run Growth Analysis
```bash
python .claude/skills/growth-analyst/scripts/analyze.py {TICKER} {END_DATE}
```

## Metrics Analyzed

### Revenue Growth
- YoY growth rate
- Growth acceleration/deceleration
- Consistency over multiple periods

### Earnings Growth
- EPS growth trajectory
- Earnings quality
- Operating leverage

### Margin Expansion
- Gross margin trends
- Operating margin improvement
- Net margin trajectory

### Growth Sustainability
- Return on capital
- Reinvestment rate
- Competitive position

## Signal Interpretation
- **Bullish**: Accelerating growth, expanding margins, sustainable trajectory
- **Neutral**: Stable growth, flat margins
- **Bearish**: Decelerating growth, margin compression

## Example
```
Analyze NVDA as of 2024-12-01 for growth characteristics.
```
