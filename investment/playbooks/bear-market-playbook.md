# 熊市心理预案（Bear Market Playbook）

> 15-20 年周期必然经历 2-3 次熊市（历史数据）。
> 本文件是"提前写好"的弹药库 —— 当大盘指数跌 -15% / -20% / -30% / -40% 时 Claude 自动调用。
> 目的：防止"放弃 DCA"这个唯一真正致命的失败模式。

---

## 为什么需要这个

**Dalbar 研究：** 普通投资者实际年化 = 基金年化一半。**差距全在行为**：熊市恐慌清仓、牛市追高。

**当前版本设计：** 60% 股 + 30% 黄金/现金 + 分散 = 熊市 -23% 到 -28% 区间。**这是设计内的，不是失败。**

**真正失败的路径：**
- 看到账户 -25% 吓停定投 → 错过反弹
- 看到账户 -30% 清仓避险 → 锁定亏损
- 看到账户 -40% 恐慌换小盘股赌反弹 → 二次伤害

**本 playbook 就是防这 3 件事。**

## Claude 在不同回撤时的强制响应

### 回撤 -10% 到 -15%（小回调）

**用户可能说：** "跌了 10% 呢"、"要不要先观望下个月"

**Claude 响应模板：**
```
历史上大盘指数每年都会回撤 5-15%，这是正常波动。
当前版本设计的风险容忍（IPS 第 3 节）是单年 -30%。
当前 [-X%] 远未触及任何触发条件。
按计划 25 日定投，不择时。
```

### 回撤 -15% 到 -20%（进入熊市区域）

**用户可能说：** "是不是要暂停定投"、"别人都说要避险"

**Claude 响应模板：**
```
历史参考：
- 2022 年标普 -19%，当前版本同期模拟 -23%
- 9 个月内恢复
- 坚持 DCA 的人获得便宜筹码

当前版本设计允许 -23% 回撤（IPS 第 3 节）。当前 [-X%] 属于设计范围内。

现金仓档位启动（portfolio-config.txt 档位 2）：
- 累计现金仓的 25% 额外投入核心 ETF
- 本月现金储备照常积累

按计划 25 日定投，并执行档位 2 加仓。
```

### 回撤 -20% 到 -30%（熊市确认）

**用户可能说：** "真的要跌到归零吧"、"这次不一样"、"我要清仓"

**Claude 响应模板：**
```
这就是"真正检验"的时刻。

回顾你建仓时的 thesis（decision-journal 记录）：
- 当时已预见 2022 模拟回撤 -23%
- 当时明确接受"积累期高风险承受"
- 当时的长期目标就是熬过这种时期

历史参考（剔除情绪后的真实数据）：
- 2008 金融危机：大盘 -37%，18 月恢复，坚持 DCA 者 2013 年跑赢一倍
- 2020 COVID：大盘 -34%，5 个月恢复
- 2022 加息：大盘 -19%，2024 年新高

当前 [-X%]，现金仓档位 3 触发：
- 再用 25% 现金：核心 ETF 70% + 单股 30%
- 本月现金储备继续积累
- 定投日照常执行

**"这次不一样" 是 1929/2000/2008/2020 每次都有人说的话。每次都错。**

清仓 = 锁定亏损 + 错过反弹 + 15 年复利归零。

按计划执行，不清仓，不择时。
```

### 回撤 -30% 到 -35%（接近警戒线）

**用户可能说：** "要不要拿 T4 现金救命"、"至少减仓吧"

**Claude 响应模板：**
```
IPS 第 3 节警戒线是 -35%（累计峰谷）。当前 [-X%]，接近但未突破。

现金仓档位 4 触发：
- 动用剩余全部现金仓
- 核心 ETF 60% + 单股 40%（穿透单股若超上限则全 ETF）
- 暂停黄金加仓

**这是最好的买点，不是最坏的时点。** 2009-03、2020-03 都是在最恐慌时触底。

按计划继续定投，档位 4 现金全部投入。

**不减仓。不清仓。** 减仓 = 用最低点卖 = 永久损失。
```

### 回撤 > -35%（突破警戒线）

**用户可能说：** "超过你说的 -35% 了，现在怎么办"

**Claude 响应模板：**
```
IPS 第 3 节警戒线突破。按规则进入"策略全盘复盘"流程。

**注意：复盘 ≠ 清仓，是检讨策略是否失效。**

立即行动：
1. 启动紧急 stock-audit（每只持仓）
2. 启动 IPS 审查（是否有系统性假设错误）
3. 评估是历史尾部事件还是结构性崩溃

**大概率结论仍是"继续 DCA"。** 历史上大盘跌 >-40% 的场景：
- 1929-1932 大萧条（-86%）→ 20 年恢复
- 2007-2009 金融危机（-56%）→ 4 年新高
- 即便最坏情况，DCA 策略长期仍跑赢一次性清仓 + 后续现金

本月定投继续执行。复盘在 24 小时内完成。
```

## 触发后 Claude 必做的事

当用户提到"回撤"/"跌"/"清仓"/"暂停"等词，Claude 必须：

1. **读 records/decision-journal.md 建仓 + 早期决策两条**（引用当时 thesis）
2. **查当前大盘距 ATH 实际跌幅**（不凭用户感受）
3. **查当前组合实际跌幅**（不凭用户感受）
4. **读 ips.md 第 3 节**（风险容忍阈值）
5. **按上述跌幅对应的模板响应**
6. **追加 decision-journal 记录当时用户的情绪 + 自己的响应**

## 用户可能的"破规则请求"及标准拒绝

| 请求 | 错在哪 | 标准拒绝 |
|---|---|---|
| "暂停定投 3 个月等等" | 择时 + 错过底部 | "IPS 要求不中断。历史数据显示暂停者长期跑输 5-10%" |
| "清仓避险" | 锁定亏损 | "清仓 = 把浮亏变实亏 + 错过反弹。拒绝执行。" |
| "换成债券 ETF 避险" | 熊市尾部债券也跌 | "2022 股债双杀，不是避险。维持当前版本。" |
| "加杠杆抄底" | Kelly 外 + 可能爆仓 | "杠杆在熊市=灭顶。拒绝。" |
| "清一半留一半" | 妥协 ≠ 理性 | "折中不是理性，是两边都错。维持当前版本。" |
| "至少某只跌太多减一下" | 单股情绪化决策 | "除非触发特别条款，否则不减。" |
| "换更'安全'的股（如 KO/PG）" | 换股是择时的另一种形式 | "当前版本设计时已经评估过防御型。不换。" |

## Pre-mortem：3 年后如果方案失败

**必想的 5 种失败模式：**

### 模式 1：用户在第一次 -25% 时清仓
**防护：** 本 playbook + Claude 强制响应模板
**后果：** 锁定亏损 + 15 年复利归零
**概率（按历史）：** 30%

### 模式 2：连续小回撤导致逐月"暂停定投"
**防护：** monthly-closure Step 3 强制跑触发检查 + 决策日记记录所有偏离
**后果：** 实际年化只有理论 60%
**概率：** 40%

### 模式 3：单只持仓回撤 -70% 时不减
**防护：** portfolio-config 单股特别条款（如营收连续 2 季 <10% 触发红灯）
**后果：** 穿透暴露 <X>% × -70% = 组合 -<Y>%（已预估）
**概率：** 20%（行业泡沫破的话）

### 模式 4：黄金 +50% 时清掉黄金追科技
**防护：** 再平衡规则（±5% 偏离增量校正，不卖出）
**后果：** 失去对冲，下次熊市裸奔
**概率：** 30%（若黄金大涨周期）

### 模式 5：W-8BEN 忘续签，红利扣 30% 而非 10%
**防护：** CLAUDE.md SOP 日历、tax-compliance skill、contingency E2
**后果：** 年损失 1-2%
**概率：** 10%

**每次季度扫描时复核这 5 种模式是否在发展。**

## 引用材料（Claude 熊市时必须读）

熊市响应时 Claude 必须打开以下文件作为"弹药"：

1. `records/decision-journal.md` — 建仓时的自己（非当下情绪的自己）
2. `ips.md` 第 3 节 — 风险容忍阈值（-35% 才是警戒）
3. `ips.md` 第 11 节 — 应急场景
4. `portfolio-config.txt` 现金仓档位（1-4）
5. 本文件 — 标准响应模板
6. `performance.md` 2b 节 — Benchmark 对标（确认跑输大盘是设计内）
7. `portfolio-versions.md` 当前版本段 — 组合的设计理由

**缺任一 = 响应不完整。**
