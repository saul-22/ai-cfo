## 5b. 数据核对规则（防长期不一致）

每月闭环后必须核对 3 点：
1. **持股一致性**：holding-summary 持股 = ledger 累计买入 - 卖出
2. **成本一致性**：holding-summary 成本均价 = 累计投入 / 累计持股
3. **市值一致性**：performance 当前市值 = holding-summary 当前市值

**关键计算公式：**
```
当前市值 = Σ(持股数 × 美元价) × 汇率 + 现金余额
成本均价 = 累计美元投入 / 累计持股
月度回报 = (月末市值 - 月初市值 - 当月新投入) / 月初市值
累计回报 = (当前总资产 - 累计总投入) / 累计总投入
α vs SPY = 累计回报 - SPY 同期累计回报
```

冲突时：ledger 是事实源，从 ledger 重算其他文件。

**年度对账（每年 12/31）：** 对照券商对账单核对持股数、现金余额、分红入账。差异以券商为准。
