## 如何使用

1. **复制模板文件**：`ips.md`、`investment/portfolio-config.txt` 等
2. **参考本文件的示例**，填入你自己的真实数据
3. **替换虚构代码**：`EXMP`、`DEMO` → 你的真实持仓
4. **替换虚构金额**：`5,000 CNY` → 你的真实月投入
5. **开始第一次会话**：在 Claude Code 中说"开始本月定投"

**重要**：本文件的数据全部是虚构的，仅供参考格式。
