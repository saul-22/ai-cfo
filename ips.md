## 8. Benchmark 对标

| Benchmark | 用途 | 组成 |
|---|---|---|
| **主：SPY**（或本地大盘）| 大盘对比 | 100% S&P 500 / 本地大盘指数 |
| **次：Policy Portfolio** | 剔除选股 α 后的同配置纯被动版本 | <X>% SPY + <X>% 对冲资产 + <X>% 现金等价 |

**为什么有两个：**
- vs SPY：策略整体是否跑赢大盘
- vs Policy：选股决策是否增值（剔除资产配置效应）

**对标频率：**
- 月度：记录每月超额
- 季度：累计 α 分析（Brinson 归因）
- 年度：归因分析 + 对照成功标准

**成功标准：** 5 年+ 累计 ≥ Policy Portfolio + 1% 年化。否则就该考虑改用被动策略。

## 11. 应急场景

| 场景 | 处理 |
|---|---|
| 失业 ≤ 3 月 | 继续定投（动用生活备用金）|
| 失业 3-6 月 | 暂停 T2+T3，只 T1 最低定投 |
| 失业 > 6 月 | 完全暂停定投，不赎回 |
| 重大医疗支出 | T4 现金 → T3 → T2 → T1（规避短期税务）|
| 市场 -30% | 暂停 T3，T4 现金全投 T1 |
| 市场 -50% | 按现金仓档位最深档执行 |
| 离婚 / 重大家事 | 触发 IPS 全盘复审 |
| 身故 | 项目 README 标注位置 + 配合遗产税规避 |
